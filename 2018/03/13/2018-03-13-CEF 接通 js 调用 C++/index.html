CEF 接通 js 调用 C++

CEF 接通 js 调用 C++

需要在cef reander进程下进行绑定。

重写一个CefV8Handler类的 Execute 方法

js 调用 C++  则会在Execute 方法中体现出来，所以js 调用C++ 的后续工作后Execute 里面来对接。
在render进程里面的 OnContextCreated 方法里面进行C++方法的绑定。
绑定的方法是 通过context拿到全局CefV8Value  创建自己实现的V8Handler对象在全局CefV8Value上进行绑定。SetValue (  方法名称【字符串】, CefV8Value::CreateFunction( 方法名称【字符串】, V8Handler【自己的子类】, 属性选项【V8_PROPERTY_ATTRIBUTE_READONLY】 )  )一般render进程会有代理来处理，所以OnContextCreated  写在代理里面即可。
